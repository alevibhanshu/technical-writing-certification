# Food Ordering App – User Guide

## 1. Introduction
The Food Ordering App allows users to browse restaurants, order food, and track deliveries in real time. This guide helps users understand how to use the app efficiently.

---

## 2. Prerequisites
Before using the app, ensure:
- A smartphone with Android or iOS
- Stable internet connection
- Registered mobile number or email ID

---

## 3. Getting Started

### 3.1 Install the App
1. Open the Play Store or App Store
2. Search for "Food Ordering App"
3. Tap **Install**
4. Open the app after installation

---

### 3.2 Create an Account
1. Open the app
2. Tap **Sign Up**
3. Enter:
   - Name
   - Mobile number / Email
   - Password
4. Tap **Register**
5. Verify OTP (if required)

---

## 4. Using the App

### 4.1 Browse Restaurants
1. Open the homepage
2. Use search bar or categories
3. Select a restaurant to view menu

---

### 4.2 Add Items to Cart
1. Select food item
2. Tap **Add to Cart**
3. Adjust quantity if needed

---

### 4.3 Place an Order
1. Open Cart
2. Review items
3. Tap **Checkout**
4. Enter delivery address
5. Choose payment method:
   - Cash on Delivery
   - UPI / Card / Wallet
6. Tap **Place Order**

---

### 4.4 Track Order
1. Go to **My Orders**
2. Select current order
3. View live tracking status

---

## 5. Managing Your Account

### 5.1 Update Profile
1. Go to Profile section
2. Edit name, address, or contact details
3. Save changes

---

### 5.2 View Order History
1. Open **My Orders**
2. View past orders and details

---

## 6. Troubleshooting

| Issue | Solution |
|------|---------|
| App not loading | Check internet connection |
| Payment failed | Retry or use different method |
| Order not confirmed | Contact support |

---

## 8. Conclusion
You can now browse, order, and track food easily using the app. Follow the steps in this guide for a smooth experience.
